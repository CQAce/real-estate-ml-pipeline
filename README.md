# Реализация ML-пайплайна оценки стоимости недвижимости (PropTech)

Прототип коммерческой скоринговой системы для оценки справедливой инвестиционной стоимости объектов недвижимости на основе их характеристик и гео-позиции. 
Проект разработан под требования вакансии Junior Data Scientist.

## 🚀 Основной функционал проекта
* **EDA и Очистка данных:** Автоматическая фильтрация аномалий и экстремальных выбросов в площади объектов.
* **Feature Engineering:** Написание кастомной функции расчета расстояния на сфере (формула гаверсинусов) для создания признака удаленности от центра города по сырым координатам (Широта/Долгота).
* **Машинное обучение:** Построение модели градиентного бустинга **CatBoost** с прямой обработкой категориальных текстовых признаков (районов) без ручного кодирования.

## 🛠️ Стек технологий
* **Python 3**
* **Pandas, NumPy** (обработка матриц и таблиц)
* **Scikit-Learn** (валидация данных, метрики качества MAE)
* **CatBoost** (алгоритм машинного обучения)

## 📦 Как запустить проект локально
1. Установите необходимые библиотеки:
`pip install catboost pandas numpy scikit-learn`
2. Запустите скрипт:
`python real_estate_pipeline.py`
